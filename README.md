## 你是谁、这是什么

这是我们实践团的共享工作空间。每个人在 `workspace/你的名字/` 下面有自己的工作目录，用来放你负责的材料、笔记、草稿等。

如果你正在和 AI 助手（Claude）对话，它会读到这个文件和 `CLAUDE.md`，自动了解你的角色和任务，然后指导你下一步该做什么。

## 快速上手（完全不会 Git 也没关系）

### 第一次使用

```bash
# 1. 克隆仓库到本地（只需做一次）
git clone https://github.com/lyy0323/gxs_2026_social.git
cd gxs_2026_social

# 2. 进入你的工作目录（把 xxx 换成你的拼音名）
cd workspace/xxx
```

### 每次开始工作前

```bash
# 拉取最新内容（别人可能更新了东西）
git pull
```

### 写完东西后提交

**如果你是 collaborator（林雨夜/李明睿/广寒居士/zhiyuan-251）**，直接推 main：

```bash
git add .
git commit -m "简要说明，比如：添加7月4日活动照片"
git push
```

**如果你是其他团队成员**，请创建分支再提 PR：

```bash
# 1. 创建你的分支（用你的名字，只需做一次）
git checkout -b 你的拼音名

# 2. 正常提交
git add .
git commit -m "简要说明"

# 3. 推送分支
git push -u origin 你的拼音名

# 4. 让 Claude 帮你提 PR（或自己跑下面这条命令）
gh pr create --title "简要说明" --body "详细说明"
```

> **不确定怎么操作？** 直接把你想做的事告诉 Claude，它会帮你生成具体命令。
